# Lab 6 — Advanced Vue

## Запуск

```bash
cd lab6
npm install
npm run dev
```

| Часть лабы | URL |
|------------|-----|
| **ToDo + API + Popup** | http://localhost:5176/ |
| **VeeValidate (регистрация)** | http://localhost:5176/registration.html |
| **Routing + Pinia** | http://localhost:5176/routing.html |

## Части работы

### ToDo (`/`)
Composition API, SFC, localStorage, REST `/api`, `<Popup />`, подтверждение удаления.

### VeeValidate (`/registration.html`)
Форма регистрации: email, password, критерии, чекбокс. См. `vee-validate/README.md`.

### Routing + Pinia (`/routing.html`)
Vue Router, localStorage, состояние в **Pinia**, изменения через **actions**. См. `routing/README.md`, `pinia/README.md`.
