# VeeValidate — Lab 6 (отдельная страница)

Форма **регистрации** по заданию из PDF «13. К лабе VeeValidate».

## Открыть в браузере

После `npm run dev` в папке `lab6`:

**http://localhost:5176/registration.html**

С главной ToDo: ссылка «Форма регистрации (VeeValidate)».

## Требования задания

1. Поля **email** (корректный адрес) и **password** (≥ 8 символов + сложность).
2. Красная рамка — ошибка, зелёная — ок, пустое поле без подсветки.
3. Чекбокс **I agree with license agreement** — без него кнопка неактивна.
4. Пароль: цифры, нижний/верхний регистр, спецсимволы; иначе красная рамка и неактивная кнопка.
5. Список критериев рядом с паролем: выполненный — зелёный, невыполненный — красный.

## Файлы

| Файл | Назначение |
|------|------------|
| `components/RegistrationForm.vue` | Форма регистрации |
| `schemas/registrationForm.js` | `email`, `password` |

ToDo List (`/`) использует обычную форму без VeeValidate.
