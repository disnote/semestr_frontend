Axios: GET https://vk.com в браузере

Страница подключает Axios из node_modules (после npm install). Удобнее открыть через локальный HTTP-сервер из корня проекта, например npx --yes serve ., затем перейти к этому файлу.

CORS: запрос к другому origin (vk.com) из вашей страницы разрешён только если сервер отдаёт заголовок Access-Control-Allow-Origin с вашим origin. У публичной главной VK этого обычно нет — в консоли/ниже ожидается сетевая ошибка (браузер блокирует чтение ответа), хотя сам HTTP-запрос мог уйти.
